Introduction

Merci d'avoir acheté un template WordPress. Cette documentation est composée de plusieurs parties et décrit le processus entier de paramétrage et d'administration d'un site WordPress à partir de zéro.

Qu'est-ce que le CMS WordPress?

WordPress est un outil de blogging open source et un système de gestion de contenu (Content Management System) basé sur PHP et MySQL. Il permet de créer des sites web et de puissantes applications en ligne et ne demande quasiment aucune connaissance technique. Plusieurs aspects, dont sa simplicité et son extensibilité, ont fait de WordPress le logiciel de site le plus populaire. En Apprendre Plus

Qu'est-ce qu'un template WordPress

Un template WordPress est un thème pour un site construit avec la plate-forme de CMS WordPress. En d'autres termes, on peut facilement changer l'apparence d'un site Web WordPress en installant un nouveau template en quelques simples étapes. Avec toute sa simplicité, un template WordPress est fourni avec tous les fichiers source nécessaires et vous êtes libres de le modifier ou de l'étendre à votre guise.

Structure des fichiers

Le package que vous avez téléchargé est constitué de plusieurs dossiers. Voyons ce que chaque dossier contient:

  • documentation - contient les fichiers de la documentation.
    • documentation.html - le fichier principal de la documentation. C'est ici que vous êtes :)
  • screenshots - contient des captures d'écran du template. Ne doit pas être mis en production.
  • sources.zip - contient les fichiers source du template.
    • psd - les fichiers source Adobe Photoshop du template (.psd).
  • theme - contient les fichiers du thème WordPress.
    • CherryFramework.zip - Archive du thème Cherry Framework. Contient tous les fichiers du thème. Doit être installé avec le gestionnaire d'extensions de WordPress.
    • theme####.zip - archive du thème (thème fils). Contient tous les fichiers du thème. Doit être installé avec le gestionnaire d'extensions de WordPress.
    • sample_data - contient des fichiers pouvant donner à votre site WordPress l'apparence de notre démo live.
      • sample_data.xml - contient tous les samples de données du template (articles, pages, catégories, etc.).
      • widgets.json - contient tous les paramètres des widgets.
      • plusieurs images et autres fichiers média
    • manuall_install - contient des fichiers pouvant donner à votre site WordPress l'apparence de notre démo live.
      • uploads - contient les images du thème.
      • theme#####.sql - fichier de sauvegarde de base de données (contient les éléments du thème).
  • license.txt - contient les détails sur la license.
  • fonts_info.txt - contient des liens où les polices propres au thème peuvent être téléchargées.
  • info.txt - contient des instructions sur l'extraction des fichiers source.

Validation

Les deux codes, le balisage HTML (Langage hypertexte) et les styles CSS, utilisés dans nos modèles, sont sémantiquement corrects et valides. Néanmoins, certaines erreurs W3C peuvent toujours avoir lieu. Rendre valide le code 100% W3C élimine l'utilisation des technologies modernes de site web comme les fonctions de CSS3 et des balises HTML5. Notre objectif est de fournir une riche expérience de l'utilisateur avec des modèles de haute qualité et parfois nous devons enfreindre certaines règles.

Installation du template

L'installation d'un template n'a jamais été aussi simple. Le processus d'installation ne comporte que trois étapes simples: l'installation de Cherry framework, l'installation du thème fils et l'installation des données du sample.

Toutes les étapes de l'installation peuvent être faites depuis l'interface WordPres. Ça rend le processus aisé et rapide.

Vous pouvez vérifier ça en cliquant sur le lien ci-dessous:

Étape 1. Installation de CherryFramework

  1. Connectez-vous au panneau d'administration de WordPress (ajoutez /wp-admin à votre nom de domaine dans la barre d'adresse de votre navigateur).
  2. Accédez au menu Apparence > Thèmes et cliquez sur le boutonAjouter.
  3. Cliquez sur le lien Mettre un thème en ligne.
  4. Cliquez sur le bouton Parcourir et naviguez jusqu'au template décompressé sur votre ordinateur.
  5. Cherchez un fichier nommé CherryFramework.zip. Cliquez dessus pour le selectionner, puis cliquez sur OK.
  6. Cliquez sur le bouton Installer Maintenant puis attendez que l'installation se termine.
  7. Cliquez sur le lien Activer le thème.

Pour plus d'informations, n'hésitez pas à consulter le tutoriel vidéo détaillé sur Comment installer le template Cherry framework.

Félicitations, vous venez d'installer le framework.

Étape 2. Installation du thème

Vous pouvez maintenant procéder à l'installation du thème fils: theme####.zip. Les étapes sont les mêmes:

  1. Allez sur le menu Apparence > Thèmes et cliquez sur le bouton Ajouter.
  2. Cliquez sur le lien Mettre un thème en ligne.
  3. Cliquez sur le bouton Parcourir et naviguez jusqu'au template décompressé sur votre ordinateur.
  4. Cherchez un fichier nommé theme####.zip (avec XXXX correspondant au numéro unique de votre thème). Cliquez dessus pour le sélectionner, puis cliquez sur OK.
  5. Cliquez sur le bouton Installer Maintenant et attendez que l'installation se termine.
  6. Cliquez sur le lien Activer pour que l'apparence de votre site soit celle du thème que vous avez acheté.

Félicitations, le framework et le thème ont été installés avec succès!

Étape 3. Installation des Données du Sample

Pour que votre site WordPress ressemble à notre démo live, veuillez suivre les instructions ci-dessous. Nous suggérons également de faire un fichier de sauvegarde .sql. exporté depuis votre base de données.

NB: Pour les templates conçus avec Cherry Framework v3.0 ou plus antérieur, pour installer les données du Sample vous devez uploader les fichiers XML et JSON du dossier "sample_data". Les images et autres fichiers média doivent être uploadés depuis le dossier "manual_install\uploads". Assurez-vous d'avoir uploadé toutes les images de tous les sous-répertoires.
  1. Allez au menu Cherry Plugin > Import Content.
  2. Tout d'abord, vous pouvez voir le tableau des nécessités qui indique si votre hébergeur est conforme aux besoins de Cherry Plugin. Veuillez contacter votre hébergeur s'il y a le moindre avertissement.
  3. Si tous les paramètres du serveur sont OK vous verrez l'écran d'upload. Ouvrez le répertoire "theme/sample_data" du package du template et uploadez tous les fichiers sur WordPress. Vous pouvez uploader tous les fichiers en les selectionnant dans le répertoire "theme/sample_data" (CTRL+A ou Command+A) et en les glissant dans la zone d'upload.
  4. Assurez-vous d'avoir uploadé tous les fichiers du répertoire sample_data, y compris les fichiers .JSON et .XML. Une fois l'upload achevé, cliquez sur le bouton "Continue Install" pour terminer.

    ATTENTION: Les fichiers .JSON et .XML sont nécessaires pour l'installation des données du sample. Le bouton Continue Install sera actif seulement si les fichiers .XML et .JSON ont été uploadés.
  5. Un clic sur le bouton "Continue Install" lancera le processus d'installation des données du sample. Les étapes de d'installation peuvent prendre du temps selon les paramètres de votre serveur. Soyez patient.
  6. Félicitations! Vous avez installé votre Thème Cherry WordPress correctement. Vous pouvez voir le thème maintenant.

Prise en Main

Préparation

Avant de paramétrer votre site WordPress, assurez-vous d'être complètement préparé. Suivez les étapes de préparation suivantes:

Logiciel d'édition

Pour être à l'aise en travaillant avec votre template WordPress, nous recommandons de télécharger toutes les applications requises. Vous pouvez voir la liste des logiciels requis sur la page de prévisualisation du template.
Les besoins peuvent varier selon le template, nous vous dirons en général ce qui est nécessaire:

  1. Tout d'abord, il vous faut les bonnes applications pour extraire les sources de l'archive protégée par mot de passe sources_#########.zip. Vous pouvez utiliser WinZip 9 et + (Windows) et Stuffit Expander 10 et + (Mac).
  2. Vous pouvez également avoir besoin de l'application Adobe Photoshop. Elle est utilisée pour éditer les fichiers source .PSD, ce qui est nécessaire si vous voulez modifier le design et les images du template.
  3. Pour modifier les fichiers du code source du template, il vous faut un éditeur de code comme Adobe Dreamweaver, Notepad++, Sublime Text, etc.
  4. Pour uploader des fichiers vers le serveur d'hébergement, vous pouvez avoir besoin d'un Gestionnaire de FTP comme Total Commander, FileZilla, CuteFTP, etc.

Hébergement

Comme le CMS WordPress est une application basée sur PHP/MySQL, vous devez préparer un environnement d'hébergement pour accueillir WordPress.

Si vous avez un hébergement, assurez-vous qu'il est conforme aux Besoins logiciels de WordPress et qu'il est prêt à être utilisé pour les sites WordPress.

Autrement, vous pouvez avoir WordPress localement sur votre ordinateur à l'aide d'un serveur local. Pour créer un serveur d'hébergement local, utilisez les applications localhost telles que WAMP, AppServ, MAMP, etc. Elles peuvent toutes être installées facilement comme tout autre programme et utilisées pour faire tourner WordPress.

Installation Manuelle

Si vous avez des problèmes pour installer les données du sample, veuillez suivre les instructions ci-dessous.

ATTENTION: Importer le fichier SQL dans votre base de données écrasera le contenu existant et les paramètres de votre site. N'importez PAS le fichier SQL si vous voulez conserver le contenu existant.
NB: Faites toujours une sauvegarde de votre base de données avant de faire toute modification.
Notez que le préfixe des tables de la base de données WordPress doit être wp_.

Pour que votre WordPress ressemble à notre démo live, veuillez suivre les instructions ci-dessous.

  1. Assurez-vous que le thème a été installé en suivant les instructions suivantes.
  2. Copiez le dossier "uploads" (qui est dans le dossier theme/manual_install/ du package de votre template) dans le dossier /wp-content/ de votre FTP.
  3. Ouvrez le fichier theme####.sql qui est dans le dossier theme/manual_install/ avec n'importe quel éditeur de texte (de préférence Dreamweaver ou Notepad) et remplacez toutes les instances de "your_website_url_here" par l'URL de votre site dans le document en entier à l'aide de l'outil Find and Replace (faites le raccourci Ctrl+H pour ouvrir cette fenêtre). Par exemple: http://www.monsite.com
  4. Assurez-vous qu'il n'y a pas de signe slash "/" à la fin de l'adresse et que l'url commence bien par http://www.
  5. Enregistrez vos modifications et fermez le fichier.
  6. Vous pouvez maintenant importer le fichier dump avec l'outil phpMyAdmin ou tout autre outil de gestion de base de données.

S'il y a des problèmes lors de l'installation via le panneau d'administration, voici l'autre moyen de le faire.
Installer les Données du Sample via FTP:

Pour utiliser cette méthode, vous devez avoir accès aux fichiers de votre site WordPress par FTP avec le Gestionnaire de Fichiers du panneau de contrôle de votre hébergement ou un client FTP comme Filezilla, CuteFTP, Total Commander, etc.

Si vous avez installé le thème avec les données du sample ou manuellement et le thème apparait autrement que sur la démo Live, ouvrez les Options de Cherry et cliquez sur "Restore Defaults."
  1. Décompressez les fichiers CherryFramework.zip et theme####.zip dans n'importe quel dossier de votre disque dur (faites d'abord un clic droit sur chaque fichier .zip, selectionnez Unzip to… CherryFramework et theme#### respectivement, afin d'avoir les deux dossiers nommés CherryFramework et theme####).
  2. Uploadez les dossiers CherryFramework et theme#### vers le dossier /wp-content/themes/ sur votre serveur FTP.
  3. Connectez-vous au panneau d'administration de WordPress (ajoutez /wp-admin après votre nom de domaine dans la barre d'adresse du navigateur).
  4. Allez sur le menu Apparence > Thèmes.
  5. Dans la section Thèmes Disponibles trouvez theme#### et activez-le en cliquant sur le bouton Activer.

Impossible de charger Google Fonts:

Ouvrir ...\CherryFramework\admin\options-sanitize.php fichier

Recherchez la ligne: 

wp_enqueue_style( "options_typography_$f", "//fonts.googleapis.com/css?family=$f$ch", false, null, 'all' );

Remplacez-le par ce qui suit:

wp_enqueue_style( "options_typography_$f", "//fonts.useso.com/css?family=$f$ch", false, null, 'all' );

Sauvegarde et Mise à Jour

Avant de mettre à jour Cherry Framework, nous recommadons fortement de faire une sauvegarde de la version actuelle.

Sauvegarde

Vous pouvez créer une sauvegarde à la fois pour CherryFramework et pour le thème fils. Ceci peut se faire dans la section "Cherry Options > Data management" du panneau d'administration de WordPress.

La sauvegarde peut se faire facilement en cliquant sur les boutons "Backup" sous les blocs Cherry Framework et Child Theme.
Vous pouvez maintenant trouver l'archive avec votre framework ou votre thème fils dans le dossier /wp-content/themes_backup/.

Vous pouvez aussi la télécharger en cliquant sur le bouton "Download backup". Si vous rencontrez des difficultés, vous pouvez réinstaller le framework dans Apparence -> Thèmes -> Ajouter des thèmes -> Mettre un thème en ligne.

Mise à jour

Pour mettre à jour Cherry Framework, il faut faire ce qui suit:

  1. Dans votre panneau d'administration, ouvrez l'onglet "Cherry Options > Data management".
  2. Cliquez sur le bouton Update sous le bloc Cherry Framework
  3. Attendez que le processus commence.
  4. Détendez-vous et Cherry Framework fera tout tout seul.

Article au Format Standard

Un article normal, standard dispose des champs suivants:

  • Titre de l'Article - titre de l'article
  • Contenu de l'Article - contenu de l'article
  • Catégories - catégorie(s) de l'article
  • Mots-clés - mots-clés de l'article
  • Image à la Une - image de l'article
  • et d'autres champs pouvant être activés dans la section Options de l'écran située en haut à droite.

Format Standard d'Article

Article au Format Simplifié

C'est une version simplifiée d'article normal. En d'autres termes, il y a seulement un champ de contenu d'article et vous n'avez pas à entrer un titre d'article ni à remplir les autres champs. C'est ce qu'on appelle un post-note.

Format Simplifié

Article au Format Gallerie

La particularité de ce format d'article est la possibilité d'avoir un nombre illimté d'images.

Ajout d'un Article au Format Gallerie:

  1. Cliquez sur le bouton Add Media.
  2. Pour ajouter des images depuis votre disque dur, glissez et déposez vos fichiers dans la boîte qui apparait ou cliquez sur Select Files pour choisir sur votre ordinateur une image à uploader.
  3. Une fois les images uploadées, vous verrez toutes les images dans votre Bibliothèque Média. Sélectionnez Uploaded to this post pour voir les images jointes à cet article.
  4. Après avoir fini vos modifications, vous pouvez fermer la Bibliothèque Média. Les images uploadées seront automatiquement jointes.

Le format gallerie a tous les champs standards (comme le titre, le contenu, les catégories, etc.) à l'instar du du format d'article traditionnel.

Article au Format Lien

Article au Format Lien

Après avoir choisi le format Lien, vous obtiendrez un champ URL supplémentaire où vous devez préciser le lien (URL) de la page/ressource ciblée. Le titre de l'Article sera utilisé comme le texte de votre lien.

Si vous remplissez le champ de contenu de l'article, il servira de description de la page/source ciblée.

Article au Format Image

Article au Format Image

Un article au format Image est un article dont le format est basé sur une seule image uploadée via l'option Image à la Une.
Après avoir choisi le format Image, vous pourrez désactiver la fonction image en lightbox en sélectionnat le paramètre correspondant pour définir l'option Enable Lightbox.

Article au Format Citation

Article au Format Citation

Un article au Format Citation permet comme son nom l'indique de citer un article. Il sert à publier des citations.

En choisissant ce format d'article, vous obtiendez un champ additionnel The Quote - le corps de la citation ainsi qu'un champ Auteur - le nom de l'auteur de la citation.

Article au Format Audio

Article au Format Audio

Le Format Audio est utilisé pour publier du contenu audio.

En choisissant ce format d'article, vous obtiendrez les options supplémentaires suivantes:

  • Title - un titre de piste
  • Artist - l'interprète d'une piste audio
  • Audio Format - un format de piste audio
  • Audio URL - un lien direct vers une piste audio

Article au Format Vidéo

Article au Format Vidéo

Un Article au Format Vidéo sert à publier du contenu vidéo.

En sélectionnant ce format d'article, vous obtiendrez les options supplémentaires suivantes:

  • Title - un titre de clip vidéo
  • Artist - un interprètre de clip vidéo
  • URL #1 - un lien direct vers un clip vidéo au format m4v
  • URL #2 - un lien direct vers un clip vidéo au format ogv
  • Embedded Code - utilisé pour exporter la vidéo hors du site, par exemple via youtube, vimeo, etc.

Les slides sont essentiellement un type d'article personnalisé ce qui veut dire qu'ils sont différents des articles ordinaires, comme les articles de blog. Ceci permet de gérer le contenu du site très facilement.

Création d'un article slider

  1. Connectez-vous sur votre Panneau d'Administration WordPress (Tableau de Bord).
  2. Cliquez sur l'onglet Slider.
  3. Maintenant cliquez sur l'onglet Add New.
  4. Insérez le titre de votre slider.
  5. Uploadez votre image avec l'option Featured Image (cliquez sur le lien Set featured image).
  6. Vous pouvez utiliser les champs suivants pour votre slider:
    • Caption - le champ pour la légende du slide.
    • URL - le champ pour l'URL du slide (vous pouvez mettre un lien externe ici).
  7. Une fois que c'est fait, cliquez sur le bouton Publish.

Description: WordPress 3.6 et + supporte aisément les vidéos YouTube et Vimeo embarquées, si vous voulez lire un fichier vidéo ou audio comme un fichier MP3 ou MP4 qui a été uploadé sur un serveur, vous devez mettre en place une sorte de lecteur pour que le fichier apparaisse comme un média plutôt qu'un lien sur votre site. En Lire Davantage

Le processus est similaire à l'insertion d'une image dans le contenu d'un article:
  1. Ajouter ou modifier une page ou un article.
  2. Cliquez dans le contenu, à l'endroit où vous voulez que le lecteur apparaisse, puis cliquez sur Add Media au-dessus de l'éditeur de contenu.
  3. Cliquez sur Upload Files.
  4. Uploadez un fichier audio ou vidéo.
  5. Assurez-vous que le fichier que vous avez uploadé est sélectionné, pui dans Attachment Display Settings, mettez Embed or Link à "Embed Media Player."
  6. Cliquez sur Insert into page.

WordPress va insérér un shortcode pour votre fichier audio ou vidéo qui sera remplacé par le lecteur.

Vous avez peut-être uploadé un fichier audio ou vidéo sur un autre serveur, comme Amazon S3 ou sur un service d'hébergement de fichiers. Entrez juste l'URL de votre fichier média, seul sur une ligne. Par exemple, vous voulez embarquer une vidéo YouTube. Collez-la juste dans le contenu toute seule sur uine ligne et le lecteur YouTube s'affichera à sa place lors de la publication. Assurez-vous que l'URL n'est pas elle-même un lien cliquable.

Politique de Cookies

Les cookies sont de petits fichiers texte que les sites placent sur les ordinateurs des visiteurs. Ils sont généralement utilisés pour identifier le visiteur et lui offrir une meilleure expérience. Les cookies sont un élément crucial pour plusieurs fonctionnalités indispensables d'un site.

Par exemple, ce sont habituellement les cookies qui permettent aux sites d'achat en ligne de se souvenir des éléments que vous avez dans votre panier. Ils servent également à vous garder connecté à un site ou à fournir d'importantes statistiques et informations sur l'utilisation aux propriétaires de sites.

Si vous possédez un site web, il utilise quasiment à coup sûr des cookies. Leur but principal est probabement de suivre le nombre de visiteurs et leur comportement avec des outils comme Google Analytics. Ils peuvent également servir à afficher des publicités pertinentes aux visiteurs, ou - si vous vendez en ligne - à mettre en avant des éléments clés de votre système d'achat en ligne.

  1. Dans votre panneau d'administration WordPress, allez dans le menu Extensions -> Ajouter.

  3. Cliquez sur Parcourir.
  4. Sélectionnez un des fichiers plugin dans ce répertoire et cliquez sur OK.
  5. Cliquez sur Installer Maintenant.
  6. Une fois le plugin installé, cliquez sur Activer.
