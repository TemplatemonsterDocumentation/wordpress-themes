Introduzione

Grazie per aver acquistato un nostro template per Wordpress. Questa documentazione è suddivisa in diverse sezione e illustra l'intera procedura per creare e gestire un sito Wordpress a partire da zero.

Cos'è il CMS WordPress?

WordPress è uno strumento gratuito e open source per la creazione di blog, e un Sistema di Gestione Contenuti, "Content Management System" (CMS) basato su PHP e MySQL. Esso ti permette di costruire siti e potenti applicazioni web e non richiede capacità o competenze tecniche particolari. Molte delle sue caratteristiche, soprattutto la sua facilità d'uso e la sua espansibilità, lo hanno reso il più popolare software per siti web sul mercato. Altre Informazioni.

Cosa è un Template WordPress

Un Template WordPress è la struttura di base di un sito web costruito con la piattaforma WordPress. In altre parole, installando un nuovo template e seguendo pochi semplici passi potrete facilmente modificare l'aspetto del vostro sito WordPress. Con tutta la semplicità che lo caratterizza, un template WordPress include anche tutti i file di origine necessari, che voi potrete anche modificare o espandere a seconda delle vostre esigenze.

Struttura dei File

Il pacchetto template che avete scaricato è suddiviso in diverse cartelle. Vediamo cosa ciascuna di esse contiene :

  • documentation - contiene i file di documentazione.
    • documentation.html - il file di documentazione principale. E' dove ti trovi proprio adesso :)
  • screenshots - contiene delle schermate del template. Non per la produzione.
  • sources.zip - contiene i file di origine del template.
    • psd - file di origine per Adobe Photoshop (.psd).
  • theme - contiene i file del tema WordPress.
    • CherryFramework.zip - Archivio del tema Cherry Framework. Contiene tutti i file del tema. Va installato attraverso la gestione estensioni di WordPress.
    • theme####.zip - archivio tema (tema child). Contiene tutti i file del tema. Va installato attraverso la gestione estensioni di WordPress.
    • sample_data - contiene i file che permettono di dare al tuo sito WordPress l'aspetto che vedete sulla nostra demo.
      • sample_data.xml - contiene tutti i dati campione del template (post, pagine, categorie, ecc.).
      • widgets.json - contiene le impostazioni per widget.
      • immagini varie e altri file multimediali
    • manuall_install - contiene i file che permettono di dare al tuo sito WordPress l'aspetto che vedete sulla nostra demo live.
      • uploads - contiene le immagini del tema.
      • theme#####.sql - file di backup del database (contiene contenuti del tema).
  • license.txt - contiene informazioni sulla licenza.
  • fonts_info.txt - contiene link da dove scaricare font personalizzabili per il template.
  • info.txt - contiene istruzioni su come estrarre i file di origine.

Installazione Template

Installare un template non è mai stato così facile. La procedura di installazione si divide in sole tre semplici fasi: installazione del Framework Cherry, installazione del tema child, e installazione dei dati campione.

Tutte le fasi dell'installazione possono essere eseguite all'interno dell'interfaccia WordPress. Questo rende la procedura comoda e veloce.

Puoi verificarlo cliccando sul link qui sotto:

Fase 1. Installazione CherryFramework

  1. Accedi al tuo pannello admin su WordPress (aggiungi /wp-admin al tuo nome di dominio nella barra degli indirizzi del tuo browser).
  2. Vai sul menù Appearance > Themes e clicca sul pulsante Add New Theme.
  3. Clicca sul link Upload.
  4. Clicca sul pulsante Browse e individua il pacchetto template estratto sul tuo computer.
  5. Cerca il file chiamato CherryFramework.zip. Clicca su di esso per selezionarlo, poi clicca OK.
  6. Clicca sul pulsante Install Now e attendi che l'installazione sia completa.
  7. Clicca il link Activate.

Per ulteriori informazioni, potete consultare le guide video su Come installare il template CherryFramework.

Congratulazioni, avete appena installato il framework.

Fase 2. Installazione Tema

Potete ora procedere con l'installazione del tema child: theme####.zip. I passi sono gli stessi:

  1. Vai sul menù Appearance > Themes e clicca sul pulsante Add New Theme .
  2. Clicca sul link Upload.
  3. Clicca sul pulsante Browse e individua il pacchetto template estratto sul tuo computer.
  4. Cerca il file chiamato theme####.zip (dove XXXX è il numero unico del tuo tema). Clicca su di esso per selezionarlo, poi clicca OK.
  5. Clicca sul pulsante Install Now e attendi che l'installazione sia completa.
  6. Clicca il link Activate per sostituire il layout attuale del tuo sito con il tema che hai appena acquistato.

Congratulazioni, il framework e il tema sono ora entrambi installati!

Fase 3. Installazione Dati Campione

Per dare al tuo sito WordPress lo stesso aspetto della nostra demo live, segui queste istruzioni. Ti consigliamo anche di tenere sempre un file .sql di backup esportato dal tuo database.

NOTA: Con i template creati su Cherry Framework v3.0 o versioni precedenti, per installare i dati campione sarà necessario caricare i file XML e JSON dalla cartella "sample_data". Immagini e altri file multimediali vanno caricati dalla cartella "manual_install\uploads". Non dimenticare di caricare tutte le immagini da tutte le sottocartelle.
  1. Vai sul menù Cherry Plugin > Import Content.
  2. Prima di tutto potresti vedere un elenco dei requisiti per verificare che il tuo server abbia i requisiti necessari per il plugin Cherry. In caso di necessità si potranno chiedere chiarimenti al proprio provider.
  3. Se tutte le impostazioni del server sono a posto, vedrai la schermata per l'upload. Apri la cartella "theme/sample_data" dal pacchetto del template e carica tutti i file su WordPress. Si possono caricare tutti i file selezionandoli nella cartella "theme/sample_data" (CTRL+A o Command+A) e trascinandoli sull'area upload.
  4. Assicurati di aver caricato tutti i file dalla cartella sample_data, inclusi i file .JSON e .XML. Quando l'upload è completo, clicca sul pulsante "Continue Install" per procedere.

    ATTENZIONE: i file .JSON e .XML sono indispensabili per procedere con l'installazione dei dati campione. Il pulsante Continue Install sarà attivabile solo quando i file .XML e .JSON saranno stati caricati.
  5. Cliccando sul pulsante "Continue Install" si avvierà la procedura di installazione dei dati campione. Le fasi di installazione potranno richiedere del tempo, a seconda delle impostazioni del tuo server. Sii paziente.
  6. Congratulazioni! L'installazione del tuo tema WordPress Cherry è ora completa. Adesso è possibile dare un'occhiata al tema.

Per Iniziare

Preparazione

Prima di procedere con la creazione del tuo sito WordPress, assicurati di avere tutto quello che ti serve. Per essere pronto, questi sono gli elementi essenziali:

Software per l'Editing

Per lavorare sul tuo template WordPress, ti consigliamo di scaricare tutte le applicazioni necessarie. Nella pagina di anteprima del template è possibile vedere la lista di tutto il software necessario.
I requisiti possono variare a seconda del template, perciò qui di seguito daremo solo delle indicazioni generali di ciò che potrà servirti:

  1. Prima di tutto, avrai bisogno delle applicazioni adatte per estrarre gli archivi sources_#########.zip protetti da password. Si possono usare WinZip 9+ (Windows) o Stuffit Expander 10+ (Mac).
  2. Potrebbe anche essere necessario Adobe Photoshop, per modificare i file di origine .PSD e nel caso in cui si volessero modificare immagini e design del template.
  3. Per modificare il codice sorgente dei file, potresti aver bisogno di un code editor come Adobe Dreamweaver, Notepad++, Sublime Text, ecc.
  4. Per caricare i file sul server, potrebbe essere necessario un gestore FTP, come Total Commander, FileZilla, CuteFTP, ecc.

Hosting

Dato che OpenCart è un'applicazione basata su PHP/MySQL, sarà necessario preparare correttamente l' ambiente di hosting per eseguire WordPress.

Se hai un hosting live, verifica che questo risponda ai Requisiti di sistema WordPress e che sia pronto all'utilizzo di siti web WordPress.

E' anche possibile eseguire OpenCart localmente sul tuo computer utilizzando il server locale. Per creare un server di hosting locale, puoi utilizzare applicazioni localhost come WAMP, AppServ, MAMP, ecc. Queste applicazioni possono essere installate facilmente come un normale programma ed essere utilizzato per eseguire WordPress.

Per imparare a configurare un ambiente di sviluppo locale, puoi seguire questi tutorial:

  2. Quando hai finito di scaricare dovrai estrarre i file dal motore e dal pacchetto del template. Per estrarre i file dal pacchetto ZIP del motore WordPress, puoi usare qualunque programma in grado di aprire gli archivi ZIP.

Installazione manuale

Se hai problemi a installare i dati campione, segui le istruzioni qui sotto.

ATTENZIONE: Importare il file SQL sul tuo database sovrascriverà i contenuti preesistenti e le impostazioni del sito. NON importare il file SQL se vuoi mantenere i contenuti attuali.
NOTA: prima di effettuare qualsiasi modifica è bene creare una copia di backup del database.
Nota che il prefisso della tabella di database dovrebbe essere wp_.

Per fare in modo che il tuo sito abbia lo stesso aspetto della nostra demo, segui queste istruzioni.

  1. Assicurati che il tema sia stato installato seguendo queste istruzioni.
  2. Copia la cartella "uploads" (che si trova sotto la cartella theme/manual_install/ del tuo pacchetto template) nella cartella /wp-content/ del tuo FTP.
  3. Apri il file theme####.sql che si trova nella cartella theme/manual_install/ usando qualsiasi text editor (preferibilmente Dreamweaver o Notepad) e sostituisci tutte le occorrenze di "your_website_url_here" con l'indirizzo del tuo sito web, in tutto il documento. Per fare prima, puoi usare la funzione Trova e Sostituisci, ("Find and Replace") (premi Ctrl+H per aprire lo strumento). Ad es.: http://www.mywebsite.com
  4. Assicurati di non avere alcun simbolo barra "/" alla fine dell'indirizzo e che l'indirizzo inizi con http://www.
  5. Salva le modifiche e chiudi il file.
  6. Ora puoi importare il file dump con lo strumento phpMyAdmin o con un altro strumento di gestione database.

Se hai problemi con l'installazione dal pannello admin, puoi provare questo metodo alternativo.
Installa i Dati Campione via FTP:

Per usare questo metodo è necessario avere accesso ai file del tuo sito WordPress su FTP attraverso il File Manager del tuo pannello di controllo di hosting, oppure con un client FTP come Filezilla, CuteFTP, Total Commander, ecc.

Se hai installato il tema con l'installazione dei dati campione o con installazione manuale e il tema appare diverso dalla demo, apri Cherry Options e clicca "Restore Defaults."
  1. Estrai gli archivi CherryFramework.zip e theme####.zip su una qualsiasi cartella del tuo disco rigido (prima clicca col pulsante destro su ciascuno dei file .zip, seleziona Unzip to… CherryFramework e theme####, rispettivamente, in modo che le cartelle siano direttamente nominate CherryFramework e theme####).
  2. Carica le cartelle CherryFramework e theme#### sulla directory /wp-content/themes/ nel tuo server FTP.
  3. Accedi al tuo pannello admin su WordPress (aggiungi /wp-admin dopo il tuo dominio nella barra degli indirizzi del browser).
  4. Vai sul menù Appearance > Themes.
  5. Nella sezione Available Themes, trova theme#### e attivalo cliccando sul pulsante Activate.

Informativa sui Cookie

I cookie sono dei piccoli file di testo che un sito web crea sul computer di un visitatore. Vengono utilizzati tipicamente per identificare quello specifico visitatore allo scopo di fornirgli un'esperienza di navigazione migliore. I Cookie sono un elemento importantissimo per il corretto funzionamento di tante funzioni dei siti web che tutti ormai diamo per scontate.

Ad esempio, sono solitamente proprio i cookie che permettono ai negozi online di ricordare quali articoli hai messo nel tuo carrello. Sono usati anche per tenere attivo il tuo login sul sito, o per dare ai gestori del sito informazioni statistiche importanti.

Se gestisci un sito web, esso quasi sicuramente usa i cookie. La loro funzione più probabile è quella di monitorare il numero dei visitatori e i loro comportamenti, servendosi di strumenti come Google Analytics. Possono essere usati anche per mostrare agli utenti delle pubblicità il più possibile rilevanti, oppure - per chi vende prodotti - per far funzionare alcuni tra gli elementi più importanti del sistema di compravendita del tuo negozio online.

26 maggio 2012 - La ICO ha pubblicato una modifica all'applicazione della regolamentazione: "il consenso implicito" è ora consentito purché sia ben chiaro dove si trovano le tue informative su cookie e privacy.

Seguendo le condizioni della legge sui Cookie, potrai aggiungere moduli sufficienti sul tuo sito. Il modulo mostrerà un messaggio di avviso riguardo l'utilizzo dei cookie sul tuo sito web. Puoi usare questi: Cookie Banner, Cookie Notice

Google services closed off in China

If you have any problems with Google Fonts and scripts in China, you need to replace:

<link href='//fonts.googleapis.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>

on

<link href='//fonts.useso.com/css?family=Open+Sans' rel='stylesheet' type='text/css'>

Also you need to connect jQuery from https://code.jquery.com/ or locally.

Backup e Aggiornamento

Prima di aggiornare Cherry Framework, ti consigliamo caldamente di fare una copia di backup della versione attuale.

Backup

Puoi creare un Backup sia per CherryFramework che per un tema child. E' possibile farlo nella sezione "Cherry Options > Data management" del pannello admin di WordPress.

Il backup può essere effettuato facilmente cliccando sui pulsanti "Backup" sotto i blocchi Cherry Framework e tema child.
Ora potrai trovare l'archivio con il tuo framework o tema child nella cartella /wp-content/themes_backup/.

Puoi anche scaricare cliccando sul pulsante "Download backup". Se ci sono difficoltà, puoi reinstallare il framework dalle tabelle Appearance -> Themes -> Install Themes -> Upload.

Aggiornamento

Per aggiornare Cherry Framework, fai così:

  1. Nel tuo pannello admin, apri la scheda "Cherry Options > Data management".
  2. Clicca il pulsante Update all'interno del blocco Cherry Framework
  3. Attendere l'avvio della procedura.
  4. Cherry Framework farà il resto da sé.

Formato di Post Standard

Un post regolare, standard, contiene i seguenti campi:

  • Post title - il titolo del post
  • Post content - il contenuto del post
  • Categories - categoria/e del post
  • Tag - i tag del post
  • Featured Image - l'immagine del post
  • e altri campi che possono essere abilitati nella sezione Screen Options all'angolo in alto a destra.

Aggiungere un post formato Gallery:

  2. Per aggiungere immagini dal tuo hard disk, seleziona e trascina i tuoi file nella finestra che appare, oppure clicca su Select Files per selezionare le immagini da caricare dal tuo computer.
  3. Terminato l'upload delle immagini, ti saranno mostrate tutte le immagini nella tua Media Library. Seleziona Uploaded to this post per vedere le immagini allegate.
  4. Finita l'operazione puoi chiudere la Media Library. Le immagini caricate saranno inserite automaticamente.

Il formato post Gallery ha tutti i campi standard (titolo, contenuti, categorie, ecc.), proprio come il formato regolare.

Formato Post Link

Shortcodes

Lo shortcode è un potente strumento per la creazione di contenuti: http://codex.wordpress.org/Shortcode. Gli shortcode sono facili da usare. Prima di tutto, assicurati che la modalità di editing selezionata sia Visual.

Poi clicca sul pulsante per gli shortcode. Dopodiché, seleziona lo shortcode che desideri inserire.


Tutti gli shortcode disponibili sono suddivisi nei seguenti gruppi:

  • Dynamic - utilizzati per l'output dei contenuti (post, tipi di post personalizzati, tag, ecc.)
  • Grid Columns - varie colonne per la formazione di griglie.
  • Elements - elementi aggiuntivi per la formazione di contenuti.
  • Other - elementi aggiuntivi per la formazione di contenuti avanzati/complessi.

Altre Informazioni

E' un'operazione simile a quella per inserire un'immagine nel contenuto del tuo post:
  1. Aggiungi o modifica una pagina o post.
  2. Clicca sui contenuti, dove desideri che appaia il riproduttore, poi seleziona Add Media sopra l'editor dei contenuti.
  3. Clicca Upload Files.
  4. Carica un file audio o video.
  5. Assicurati che il file caricato sia selezionato, poi in Attachment Display Settings, nel menù Embed or Link, seleziona "Embed Media Player."
  6. Clicca Insert into page.

WordPress avrà inserito uno shortcode per il tuo file audio o video che sarà sostituito con il riproduttore.

E' possibile che tu abbia caricato il file su un server esterno, come Amazon S3 o un servizio di file hosting. In tal caso basta inserire la URL del file (unlinked) su di una riga a parte. Ad esempio, diciamo che vuoi incorporare un video YouTube. Basta che lo incolli nei tuoi contenuti su un riga a sé e il riproduttore video di YouTube apparirà in quel punto nel momento in cui pubblichi il post. Assicurati che l'URL non sia linked.

Shop Beginners Kit

Importante! Shop Beginners Kit funziona solo con Cherry Framework versioni 2.0 o superiori.
Se hai una versione precedente alla 2.0, dovrai aggiornare Cherry Framework.

L'offerta Shop Beginners Kit può essere acquistata separatamente o insieme a un tema WordPress.

Per farlo funzionare sul tuo store devi prima di tutto dovrai installare e attivare uno tra i plugin WooCommerce o Jigoshop.

Puoi scaricare i plugin qui:

Jigoshop Plugin: http://wordpress.org/plugins/jigoshop/
WooCommerce Plugin: http://wordpress.org/plugins/woocommerce/

Attenzione! Per le operazioni più comuni basta solo uno dei due plugin.
Installazione Plugin
  1. Nel tuo pannello di amministrazione WordPress, vai sul menù Plugins -> Add New.
  2. Clicca Upload.
  3. Clicca Browse.
  4. Seleziona uno dei file del plugin in questa directory e clicca OK.
  5. Clicca Install Now.
  6. Una volta installato il plugin, clicca Activate.
Se hai acquistato questa offerta insieme al tema WordPress, dovrai eseguire alcune operazioni per assicurarti che tutto funzioni correttamente.

Inoltre potrai installare widget e demo per il tuo negozio online.

Devi effettuare le seguenti operazioni:

  1. Installazione dei widget:
    • Nel menù, trova Cherry Options -> Import
    • Seleziona jigoshop_widgets.json o woocommerce_widgets.json a seconda del plugin che hai scelto. Dopodiché, premi il pulsante Show Widget Settings (puoi saltare questo passo e impostare tutti i widget necessari manualmente usando l'opzione Appearance -> Widgets).
    • Seleziona i widget che desideri installare e poi premi il pulsante Import Settings.
    • Attendi che l'installazione sia completa.
  2. Impostare i prodotti demo.
    • Seleziona il file shop_sample_data.xml e premi i pulsanti Upload file e Import (puoi saltare questo passo e farlo in seguito).
    • Nel campo Import Author, seleziona il tuo utente e premi il pulsante Import Data.
    • Dopo aver impostato i prodotti demo, premi il pulsante Finish. Ora il tuo negozio online è pronto per altre operazioni.
Se decidi di aggiungere la funzionalità negozio a un template acquistato in precedenza, dovrai aggiungere dei file aggiuntivi che garantiscano la compatibilità di Cherry Framework con i plugin WooCommerce e Jigoshop.

Prima di tutto, dovrai acquistare l'offerta "Shop Beginners Kit". Poi dovrai effettuare alcune semplici operazioni:

  1. Installa e attiva i plugin WooCommerce o Jigoshop (basta solo uno dei due).
  2. Dalla cartella theme####, copia i due file (shop.less e shop.php) nella cartella root del tema child.

Puoi anche installare widget e demo per il tuo negozio online. Vedi le istruzioni sopra.
